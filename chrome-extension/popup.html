TALK-TO-ChatGPT

To start using this extension, you need to open the ChatGPT home page. Once you are logged in, a box will display at the top right corner of your screen with a 'START' button. Click START to start talking.

This extension allows you to talk to ChatGPT with your microphone, using your browser's voice recognition API. Then, when your browser receives the reply, it is read out loud with a voice of your choosing.

• When the box's bottom border is red, your microphone is listening
• When the box's bottom border is green, the bot is talking
• Access the settings by clicking the settings icon: change voice/language, bot talking speed, and more.

If something doesn't appear to work, click here for status and troubleshooting. Having trouble getting it to work?

Is the extension no longer working for you? The box isn't appearing where it should, or something appears to be off? Make sure to check the project page for the latest status information. Sometimes OpenAI update their user interface and break the extension, but I can usually get it fixed quickly.

1. This extension is only designed for Chromium-based desktop browsers such as Google Chrome or Microsoft Edge. If it doesn't work in your specific browser, it's because your browser doesn't support the necessary APIs. There is no need to contact us to add support to another browser, we can't do anything about it. Support for these APIs must be implemented by the browser itself.

2. What to do if the computer isn't recognizing text when you speak? Did you make sure to give permission to use your microphone? Is the microphone recording volume high enough? Check this speech recognition test page. If the speech recognition isn't working on that page, then it won't work on Talk-to-ChatGPT. Also, for Mac users, make sure your laptop's lid isn't in a closed state (because of this).

3. What to do if the bot's voice is scrambled, jammed, or not working at all? Try opening the settings and changing the bot's voice to a different option. Also make sure the talking speed is set to 1. Sometimes, voices listed in the text-to-speech API don't work correctly, for some reason.

4. What if it worked well for a bit, and suddenly nothing works correctly anymore? In that case, try restarting your browser. There is sometimes a bug with Chromium's voice API, when the bug occurs, it can completely jam the API until you restart your browser.

5. Do you have another problem that isn't listed here? Please visit the project page, read the FAQ section, and if you answer still isn't in there, please ask your questions by clicking on the 'Issues' tab at the top of the page.
